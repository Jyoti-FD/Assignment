11* What is the difference between a block-level element and an inline element?

Block-level Elements

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element. A block-level element always takes up the full width available (stretches out to the left and right as far as it can). Two commonly used block elements are: p tag and div tag. The p tag element defines a paragraph in an HTML document. The div tag element defines a division or a section in an HTML document.

Inline Element

An inline element does not start on a new line. An inline element only takes up as much width as necessary. This is a element inside a paragraph.
